# AgroDiary — Журнал фермы для Android

![Version](https://img.shields.io/badge/version-1.3.0%20Alpha-orange)

Мобильное приложение для эффективного управления фермерским хозяйством. Позволяет вести учет животных, персонала, задач, запасов кормов и произведенных товаров.

## ✨ Основные функции

*   **🐄 Учет животных**: Регистрация поголовья, отслеживание статуса, породы и веса.
*   **👷 Персонал и задачи**: Управление сотрудниками и назначение задач с приоритетами и сроками.
*   **🌾 Запасы кормов**: Мониторинг остатков кормов с предупреждениями о низком уровне.
*   **🧺 Учет товаров**: Учет произведенной продукции (молоко, яйца и т.д.) и ее стоимости.
*   **📖 Журнал событий**: Ежедневная фиксация всех важных событий на ферме.
*   **📊 Отчеты**: Сводная статистика по всем модулям для быстрого анализа состояния хозяйства.
*   **🌓 Темная тема**: Поддержка светлой, темной и системной тем оформления.

## 🛠 Технологический стек

*   **Язык**: Kotlin
*   **UI**: Jetpack Compose (Material 3)
*   **Архитектура**: MVVM + Clean Architecture (упрощенная)
*   **База данных**: Room (SQLite)
*   **Инъекция зависимостей**: Hilt
*   **Навигация**: Navigation Compose
*   **Асинхронность**: Coroutines & Flow

## 🚀 Сборка и запуск

Для сборки проекта вам потребуется Android Studio (Hedgehog или новее) и JDK 17.

```bash
# Клонирование репозитория
git clone https://github.com/OPnce32/AgroDiary.git

# Сборка проекта
./gradlew build

# Запуск тестов
./gradlew test

# Сборка установочного APK
./gradlew assembleDebug
```

## 📁 Структура проекта

*   `data/`: Слой данных (Room entities, DAOs, Repositories).
*   `di/`: Настройка Hilt модулей.
*   `ui/`: Презентационный слой на Jetpack Compose.
    *   `theme/`: Определение темы и стилей.
    *   `components/`: Общие UI компоненты.
    *   `*feature*/`: Экраны и ViewModels конкретных функций.

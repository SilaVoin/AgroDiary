# Нефункциональные требования и безопасность AgroDiary

Версия приложения: 1.5.1  
См. также: [Функциональные требования](functional-requirements.md), [Архитектура](architecture.md)

## Платформа и производительность
- NF-PLAT-001 Поддержка Android 8.0+ (minSdk 26), targetSdk 34, compileSdk 34.
- NF-PERF-001 Работа offline; все операции выполняются локально (Room + Flow/Coroutines).
- NF-PERF-002 Списки должны отображаться без ощутимых задержек на типичных данных малого/среднего хозяйства (<10k записей).

## Надежность и целостность данных
- NF-REL-001 Все критичные операции CRUD выполняются транзакционно через Room.
- NF-REL-002 При ошибках сохранения показываются сообщения в UI (Snackbar); состояние форм сохраняется в памяти экрана.
- NF-REL-003 Журнал действий фиксирует операции с животными и сотрудниками для аудита.

## Безопасность
- NF-SEC-001 Пароли хэшируются PBKDF2WithHmacSHA256 с солью (EncryptedSharedPreferences для хранения соли/хеша).
- NF-SEC-002 Поддерживается апгрейд со старого SHA-256 хеша при успешном входе.
- NF-SEC-003 Активная сессия хранится в EncryptedSharedPreferences (ключ current_user_id).
- NF-SEC-004 Данные пользователей, животных и т.п. хранятся в локальной БД Room; шифрование БД не используется.
- NF-SEC-005 Внешние сети/апи не используются; данные не покидают устройство.

## Приватность
- NF-PRV-001 Приложение не собирает телеметрию и не отправляет данные наружу.
- NF-PRV-002 Фото и вложения хранятся как локальные URI, без загрузки в облако.

## UX и доступность
- NF-UX-001 Интерфейс на русском языке; подсказки и сообщения должны быть человекочитаемыми.
- NF-UX-002 Удаление сущностей с подтверждением применяется только к задачам, кормам, продуктам и записям журнала; животные и сотрудники удаляются сразу по требованию пользователя.
- NF-UX-003 Переключение темы (system/light/dark) применяется глобально через Material3.

## Эксплуатация и обновления
- NF-OPS-001 Поставляется в виде APK; установка вручную.
- NF-OPS-002 Нет фоновых служб и уведомлений; приложение не должно разряжать батарею в фоне.
